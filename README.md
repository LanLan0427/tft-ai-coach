A Discord bot that analyzes Teamfight Tactics (TFT) match history and provides humorous, AI-powered coaching feedback.

## Features
* **Match Analysis**: Fetches the latest match data (placement, units, traits) using Riot API.
* **AI Commentary**: Uses Google Gemini AI to act as a "toxic" or "humorous" coach, giving feedback on your comp.
* **Discord Integration**: Sends detailed reports directly to Discord channels.

## Tech Stack
* Python 3.11
* Riot Games API (TFT-MATCH-V1, ACCOUNT-V1)
* Google Gemini API (Generative AI)
